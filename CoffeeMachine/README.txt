This is a coded Coffee Machines that has attributes:
-Sugar ammount
-Beans amount
-Water amount
-Usage amount 

Usage amount limit is a constant which secures as how often the cofee machine should be cleaned.

Coffee machine is able to:
• Fill water / beans / sugar;
• Make three types of coffee and hot water;
• Coffee machine notifies you if it's lacking any of the products as well as informs you if it needs to be cleaned;
• Performs cleaning (sets the usage amounts to zero when cleaned)
• Gives the status of the products and number of usaged left until the cleaning.


The class that needs to be run for the program is App. The main functionality was coded under the while loop with 
the switch statement, where every case is a selection in the menu.
